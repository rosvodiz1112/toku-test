2018三陸なう　開催報告！

~ 2018三陸なう報告会 ~

2018年12月15日(土)~16日(日)に上野公園で開催された「2018三陸なう」の報告会を開きました！
　　
報告会のまとめを簡単にご紹介します。

「ありのままの三陸を、好きになって欲しいから。」

これが今年の三陸なうのテーマでした。

東京の人に、三陸を好きになってもらいたい。しかし、それは「復興のため」ではありません。

私たちは、三陸に訪れるなかで、三陸には「震災の記憶」だけでなく歴史や文化などの魅力であふれていることを知りました。

展示や工作体験、防災に関するライフハック体験などを通してお客さんに私たちが体験した三陸の魅力を伝えたい。そんなイベントを目指しました。

「子どもの笑顔に溢れる２日間を創ることができ、嬉しく思っています。」（事務局長の松本より）

また、三陸なうは多くの方々のご厚意、ご協力によって開催することができています。
この場をお借りしてご紹介したいと思います。

さて、2019年度の開催は

2019年12月14日(土)~15日(日)

を予定しております！

今後もFacebook,WEBページ共に更新していきますので、ぜひチェックしてくださいね！（R.S.）
